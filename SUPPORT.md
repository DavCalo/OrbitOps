# Support

OrbitOps is an open-source technical preview maintained as a portfolio and learning project.

- Use **Discussions**, when enabled, for usage questions and design conversations.
- Use a **bug report** for reproducible defects.
- Use a **feature request** for product proposals.
- Use private vulnerability reporting for security issues.

Before requesting help, run:

```bash
orbitops --version
./build/orbitops_sim --version
make test
make integration
```

Include the command output, operating system, Python version, compiler version, and relevant logs. Commercial support and response-time guarantees are not currently offered.
